# Transaction-Reconciliation-Tool

## Overview
This project is intended to simulate reconcilliation between two csv files using Panda to find any discrepancies in the data between the two systems

## Features
- Loads transaction from two systems 
- Matches records using transaction ID
- Identifies missing transactions 
- Flags mismatched transaction amounts
- Outputs a reconciliation summary

## Technologies used 
- Python
- Pandas

## Simple Use case
Financial teams use reconciliation tools to ensure consistency between systems during platform migration or audits. This tool uses core logic behind such reconciliation workflows
